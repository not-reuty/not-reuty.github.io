About

Hi, I'm Reuty aka not_reuty, and I'm a computer programmer based in the UK. In my spare time I like to program different game mechanics and publish my work in open source repositories like github. I hope some of my projects are useful, both for other programmers learning how to implement different mechanics, and for game designers looking to get a feel for how different mechanic parameters affect the 'feel' of these mechanics in games.

If you have questions about any of the projects or want to get in touch, message me through reddit or twitter; u/not_reuty or @not_reuty

Damage Lab

This project is all about how damage mechanics work, how status effects can be applied to enemies, and how they can interact. This project was created using the Godot Engine and is written in GDScript.

The github repository readme file contains descriptions of each of the effects that can be applied, and a quick overview of how the user interface interacts with the game objects, and how the project is structured.
