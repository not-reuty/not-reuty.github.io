Damage Lab

This project is all about how damage mechanics work, and how status effects can be applied to enemies, and how they can interact. This project was created using the Godot Engine and is written in GDScript.

The github repository readme file contains descriptions of each of the effects that can be applied, and a quick overview of how the user interface interacts with the game objects, and how the project is structured.
